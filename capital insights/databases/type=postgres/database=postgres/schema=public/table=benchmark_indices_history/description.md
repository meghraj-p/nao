|----------|-------|
| **Row Count** | 6,497 |

## Description

This table contains daily data of closing index values, PE ratio, PB ratio, and dividend yields of NIFTY benchmark indices. The 'Date' column contains the date in datetime format. There are 4 columns corresponding to each NIFTY benchmark index:
   i.   '<Index Name>_pe': Price to Earnings Ratio (PE ratio) of the index as on the date in the 'Date' column.
   ii.  '<Index Name>_pb': Price to Book Value Ratio (PB ratio) of the index as on the date in the 'Date' column.
   iii. '<Index Name>_div_yield': Dividend Yield of the index as on the date in the 'Date' column.
   iv.  '<Index Name>_closing_index_value': Closing value of the index as on the date in the 'Date' column.
